# 数组
   * 1、对数组 `{1, 2, 3, 4, 5, 6}`，循环右移 4 次，那么将得到 `{3, 4, 5, 6, 1, 2}`，如何实现这个循环右移？
   	  * 方法一 : 分配额外空间 + `memcpy`
   	  * 方法二 : 逐个右移法
   	  * 方法三 : 逆序再分别转至
   * 2、找出无序数组中的重复的数字，数组长度为 n, 元素均在范围 `0~n-1`
     * 方法一 : 分配额外数组
     * 方法二 : bitset
     * 方法三 : 先排序
     * 方法四 : 判断下标法
   * 3、不修改数组找出无序数组中重复的数字，数组长度为 `n+1`, 元素均在范围 `1~n`
     * 方法一 : `O(n^2)` 双重循环
     * 方法二 : 分配额外数组
     * 方法三 : 二分思想划分区间 + 统计区间内个数
